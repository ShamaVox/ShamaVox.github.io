Selected work

Systems built to survive production.

The through line across everything below is a generator, verifier, and repair loop that emits a reliability signal — the difference between a prototype that demos well and a system an enterprise puts in front of customers.

AI-native retail platform

Multi-agent document intelligence

Context
Unstructured vendor documents were costing commerce businesses thousands of hours of manual reconciliation. "Usually right" isn't a standard a business can operate on.
Architecture
Multi-agent extraction converting unstructured retail documents into deterministic structured models, with real-time hallucination detection, custom confidence scoring, and stateful fallback guardrails — agents verify each other, escalate on low confidence, and fail into a known state rather than a plausible wrong one.
Result
96% extraction accuracy in production with live paying merchants. Engineered a custom onboarding path around a platform certification bottleneck to protect the early sales pipeline.
Stack
Claude API, multi-agent orchestration, confidence scoring, human-in-the-loop review tooling, commerce platform integration.
Nickelytics

Edge AI telemetry for autonomous assets

Context
Converting physical moving vehicles into measurable advertising inventory, which required validating precise edge attribution rather than loose estimation.
Architecture
Built real-time location tracking, edge attribution telemetry, computer vision running at the edge, and the software foundations from absolute zero — plus the engineering organization to operate it.
Result
Scaled recurring revenue from $20K MRR to $3.5M ARR, owning product, delivery, infrastructure cost, and the org through the entire climb. I departed before the company's later exit.
Stack
Edge computer vision, real-time geospatial tracking, attribution pipeline, cloud infrastructure engineered for sublinear cost scaling.
SignAI

Real-time ASL ↔ English interpretation

Context
Sign language interpretation is an edge-processing latency problem disguised as a translation problem — and the Deaf community had no real-time option that worked without a human in the room.
Architecture
Designed a bidirectional real-time ASL–English prototype using gesture and pose modeling, low-latency inference, and edge optimization on constrained hardware.
Result
Raised $800K preseed and ran technical validation demos for Microsoft. Handed the platform off as a Deaf-led initiative — accessibility work belongs with the community it serves.
Stack
Computer vision, gesture and pose modeling, low-latency inference, bidirectional translation pipeline.
BrainEx

Secure clinical automation frameworks

Context
Moving patient data through a regulated environment demands technical speed alongside zero-compromise security. The two pull against each other on every design decision.
Architecture
Led zero-to-one engineering strategy for secure healthcare data routing, with strict privacy access controls and comprehensive automated audit logging as first-class system properties rather than compliance paperwork.
Result
Established alignment with HIPAA, HITECH, and PHI protection mandates, converting legacy manual processes into secure clinical workflows.
