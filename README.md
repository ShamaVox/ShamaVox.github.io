## About

systems, and career track record. Built as one self-contained static HTML file —
no framework, no build step, no dependencies beyond Google Fonts.

## Privacy

This site collects nothing. There is no analytics, no tracking, no cookies set by
this site, and no forms that transmit data.

Two outbound services are referenced:

- **Google Fonts** — typefaces load from `fonts.googleapis.com`, which receives a
  request from your browser. See Google's privacy policy.
- **Calendly** — the "Book a call" links open Calendly in a new tab. Anything you
  submit there is governed by Calendly's own privacy policy.

Following links to GitHub, LinkedIn, Substack, X, or Instagram takes you to those
platforms, which have their own policies.

Email sent to the address above is used only to reply and is not added to any
mailing list.
